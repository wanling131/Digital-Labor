## 一、测试类型

### 1.1 单元测试

后端接口测试使用 Node.js 内置 test runner + supertest：

```bash
cd server && npm test
```

**覆盖范围（共 26 个用例）**：
- 健康检查、登录认证（管理端/工人端）
- 组织管理、人员档案、数据看板、在岗看板
- 权限相关：菜单、角色权限、`/api/sys/feature-status` 能力开关
- 人员认证与证书：认证列表含人脸字段、证书 CRUD
- 工人端：证书列表（GET /api/worker/certificates）、打卡（POST /api/attendance/clock）
- 结算推送、统一错误格式等

### 1.2 集成测试

前后端连通性测试：

```bash
# 先启动服务
npm run dev:server
npm run dev:web

# 运行连通性测试
npm run test:connectivity
```

### 1.3 构建测试

前端构建验证：

```bash
cd web && npm run build
```

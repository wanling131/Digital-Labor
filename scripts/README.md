# 脚本目录

本目录用于放置项目级脚本，便于维护与部署。

## 已有脚本

| 文件 | 说明 |
|------|------|
| **connectivity-test.mjs** | 前后端连通性测试。请求后端 `/api/health`、`/api`、`/api/auth/login` 及可选的前端首页；需先启动后端（可选启动前端）。根目录执行：`npm run test:connectivity`。 |

## 建议用途

- **构建**：根目录已提供 `npm run build:web`，一般无需在此新增。
- **部署**：部署前检查、上传静态资源等脚本可放此处。
- **数据**：若需定期备份 `server/data/` 或执行数据迁移，可在此添加脚本并注明用法。

详细测试步骤见项目根目录 **docs/测试说明.md**。
